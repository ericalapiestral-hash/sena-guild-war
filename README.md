# 세나 리버스 길드전 도우미

세븐나이츠 리버스 길드전 공략을 위한 길드 공유용 사이트.

## 기능

- **카운터덱 사전** — 상대 방어덱 영웅으로 검색하면 카운터 공격덱과 공략 포인트를 보여줌. 직접 추가/수정 가능.
- **영웅 DB · 덱 빌더** — 등급/속성/포지션 필터, 5인 덱 저장.
- **공략 가이드** — 길드전 규칙과 팁. 길드 자체 공략 섹션 추가 가능.
- **길드원 관리** — 담당 배정 메모, 승패 기록.
- **데이터 관리** — 입력 데이터 JSON 내보내기/가져오기 (브라우저 localStorage 저장).

## 개발

```bash
npm install
npm run dev     # 개발 서버
npm run build   # dist/ 에 정적 빌드 → 아무 정적 호스팅에나 배포 가능
```

## 데이터 구조

- `src/data/heroes.json` — 영웅 로스터 초기 데이터
- `src/data/counters.json` — 방어덱→카운터덱 초기 데이터
- `src/data/guide.ts` — 공략 가이드 초기 문서
- 사용자가 사이트에서 입력한 데이터는 localStorage(`sena-guild-war:v1`)에 저장되며,
  [데이터 관리]에서 내보낸 JSON을 위 파일에 반영하면 배포본 기본 데이터로 승격 가능.
